kalshi-agent — Personal Bot

This Discord bot is operated by a single individual for personal use.
It collects no data from users other than the operator's own Discord
user ID (loaded from .env at startup) and the contents of messages the
operator sends to the bot. No data is shared, sold, or transmitted
outside the operator's own infrastructure.

Terms of Service: Use of this bot is restricted to the operator. The
bot will ignore commands from any other user.

Privacy Policy: No data collection occurs from non-operator users. All
operator interactions are logged locally for audit purposes only.
